# socket.io & webRTC 簡易聊天室(含視訊邀請)

Demo: https://profile1.jiangshuuu.com/  

## 產品功能
- 配對1對1即時聊天
- 傳送三次訊息後可以發送視訊邀請

## 環境
### 框架
- React(FrontEnd) + node(BackEnd) + MongoDB
### 部署
- Oracle + Docker + GithubAction
### 使用套件
- simple-peer
- socket.io server&client
- styled-components
- uuid
- express
- mongoose
